A fun and engaging finger painting session where children explore colors and creativity. Each child created their own masterpiece using washable finger paints. The activity focused on color recognition, fine motor skills development, and creative expression. Children learned about primary colors and how they mix to create new colors.

  • Develop fine motor skills through finger painting
  • Learn about primary and secondary colors
  • Encourage creative expression and imagination
  • Practice following simple instructions
  • Build confidence in artistic abilities
